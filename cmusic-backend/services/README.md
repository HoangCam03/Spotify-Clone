## Spotify Backend Services

This folder contains the backend split into microservices.

Services (current):
- `gateway` (simple HTTP proxy gateway)
- `api-gateway` (gateway with middleware + routing)
- `auth-service`
- `user-service`
- `catalog-service`
- `playlist-service`
- `likes-service`
- `history-service`
- `search-service`
- `admin-service`
